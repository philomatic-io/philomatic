Remember this — Philomatic capture bookmarklet

Start the local ingest service, then drag the button below to your bookmarks bar. Click it on any page to have your graph remember that source.

  1. Start the service: pnpm serve (defaults to http://127.0.0.1:4321).
  2. Drag this button to your bookmarks bar:
    📚 Remember (Philomatic)
  3. Visit any page and click the bookmark. A toast confirms the capture.
  4. Verify from the CLI against the same database: 
    pnpm philomatic list sources
pnpm philomatic list events

The bookmarklet runs in the visited page's origin, so a page with a strict Content-Security-Policy can block the request — that's the known Stage-0 limit (the extension removes it). If you run the service with --token <t>, edit TOKEN in browser/bookmarklet.js to match and re-drag.
